>[!question] What is a Tautology?
>A statement P is called a tautology or law of propositional logic if there is a set of substatements of P such that:
>
>(a) P is a propositional combination of those substatements, and
>(b) P is true for every combination of truth values that is assigned to these substatements.
>

So, one of the first observations should be that set of these substatements must be more than 1, because if the substatement is P itself, both the truth assignments to P as True and False must imply that P is True. But when P is assigned False , then well P is False.

Following the above observation, it must be clear that P cannot be an atomic proposition. It must have a connective.

>[!faq] Contradiction 
>A statement whose negation is a tautology is called a contradiction

>[!warning] Propositionally Equivalent
>Two statements P and Q are called propositionally equivalent if $P \iff Q$ is a tautology
